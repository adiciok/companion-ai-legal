# Companion AI legal pages

Public, static legal pages for Companion AI.

- Privacy policy: `https://adiciok.github.io/companion-ai-legal/privacy-policy.html`
- Account/data deletion: `https://adiciok.github.io/companion-ai-legal/account-deletion.html`
- Terms: `https://adiciok.github.io/companion-ai-legal/terms.html`
- Password reset: `https://adiciok.github.io/companion-ai-legal/reset-password.html`

Use the privacy / deletion / terms HTTPS URLs in Google Play Console. The
`reset-password.html` page is the target of the Supabase password-recovery email
link; it must be added to the Supabase **Redirect URLs** allowlist.
